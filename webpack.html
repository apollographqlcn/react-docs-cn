Webpack加载器

Edit on GitHub

您可以使用Webpack在.graphql文件上加载GraphQL查询。包graphql-tag带有一个容易设置的加载器，并具有一些优点：

  1. 不要在客户端处理GraphQL 抽象语法树
  2. 将查询与逻辑分开
